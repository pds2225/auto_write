# 지원사업별로 각각다른 서류평가기준으로 채점했을때 고득점을 받을수있게 사업계획서 요약페이지를 작성해주는 툴

    자동 생성된 AI 프로젝트 시작 폴더입니다.

    ## 포함 파일
    - AGENTS.md : AI 작업 규칙
    - PRD.md : 서비스 기획 요약
    - TASKS.md : 작업 목록
    - RULES.md : 구현/테스트 규칙
    - START_PROMPT.txt : AI에 바로 붙여넣을 시작 프롬프트

    ## 사용 순서
    1. AI 코딩 도구를 연다.
    2. START_PROMPT.txt 내용을 붙여넣는다.
    3. TASK-01부터 하나씩 진행한다.
