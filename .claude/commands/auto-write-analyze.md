# /auto-write-analyze

## 사용 목적

사업계획서를 쓰기 **전에** 무엇을 평가하고 무엇을 제출해야 하는지, 어떤 양식을 채워야 하는지 분석한다.
두 가지 모드가 있다(읽기 전용 — 원본을 수정하지 않음).

### 1) 공고문 분석 (`announcement`)
공고 파일(DOCX/PDF/HWP/TXT)을 읽어 종합 분석한다.
- **평가기준·배점** — 사업계획서 작성 방향과 `/auto-write-bizplan` 채점 기준이 된다.
- **핵심 정보** — 지원대상·지원자격·지원금액·신청마감·제출서류·지원내용·가점·유의사항.
- AI 키가 있으면 정밀 추출, 없으면 정규식 휴리스틱(마감일·금액·서류 위주).

### 2) 양식 분석 (`form`)
제출 양식(DOCX/HWP/PDF)을 읽어 **무엇을 채워야 하는지** 요약한다.
- 작성 항목 수 / 필수 항목 수, 표 개수(+필수 셀), 이미지 슬롯 수.
- PSST 4영역(문제·실현·성장·팀) 존재/누락.
- 작성 항목 목록(필수 우선).

## 입력값

- `announcement <파일>` 또는 `form <파일>` (필수): 분석할 파일 경로.
  - 지원 형식: `.docx`(직접), `.pdf`(pypdf), `.txt/.md`(직접), `.hwp/.hwpx/.doc`(변환 후, 일부 제한).
- `--text-of "<공고 본문>"` (announcement 전용): 파일 대신 직접 텍스트 전달.
- `--no-ai` (announcement 전용): AI 없이 휴리스틱만 사용.
- `--json`: 결과 JSON 출력.

> HWP 는 한글(아래아한글)로 만들어진 경우가 많아 변환이 불완전할 수 있다. 추출이 비면
> 한글에서 **다른 이름으로 저장 → .docx 또는 .pdf** 로 바꾼 뒤 다시 분석하면 정확해진다.

## 실행 워크플로우(단계)

1. 파일 존재 확인. 없으면 "실행 막힘" 보고.
2. `cd D:\auto_write\app` 후 아래 실행.
3. 결과를 표/요약으로 정리해 보고. 공고 분석이면 평가기준을 `/auto-write-bizplan` 에 넘기도록 안내.

## 호출 에이전트

- `doc-analyzer`: 공고/양식 분석·해석 전담(읽기 전용).
- 후속: 평가기준·양식 구조를 `/auto-write-bizplan` 으로 넘겨 사업계획서 생성·완성.

## 예시 명령(실제 PowerShell)

```powershell
cd D:\auto_write\app

# 1) 공고문 분석(평가기준·자격·마감·제출서류)
python analyze_docs.py announcement "C:\공고\모집공고.hwp"

# 2) 공고 본문을 직접 붙여넣어 분석
python analyze_docs.py announcement _ --text-of "지원대상: 예비창업자 ... 평가기준: 문제인식 30점 ..."

# 3) 양식 분석(작성 항목·필수입력·PSST 구조)
python analyze_docs.py form "C:\양식\사업계획서_양식.hwp"

# 4) JSON 으로 받기(다른 도구 연계)
python analyze_docs.py announcement "공고.pdf" --json

# 5) 분석한 공고로 바로 사업계획서 생성·완성
python bizplan_autopilot.py "초안.docx" --announcement-file "공고.hwp" --target-ratio 0.85
```

## 실패 시 처리

- 파일 없음/형식 오류 → "실행 막힘" 보고, 경로·형식 재확인.
- HWP 변환 실패 → 한글에서 .docx/.pdf 로 저장 후 재시도 안내(노트에 표시됨).
- 평가기준 0개 → 공고에 배점이 명시되지 않았거나 표 형태일 수 있음. 본문/별표 확인 안내.
- PDF 텍스트 0자 → 스캔본(이미지) PDF 가능성. OCR 또는 .docx 변환 안내.

## 보고 형식

첫 줄 상태(`정상 실행 확인됨` / `미검증` / `실행 막힘` / `수정 없음`(읽기 전용)).
- **공고**: 평가기준(항목·배점·총점) / 핵심정보(자격·금액·마감·제출서류·가점) / 안내노트
- **양식**: 작성 항목 수·필수 수 / 표·이미지 / PSST 존재·누락 / 작성 항목 목록
