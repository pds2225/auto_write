## 4. 구현 기능 — SubmittableFiller 엔진 (신규)

`app/auto_write/services/submittable_filler.py`

| 메서드 | 역할 |
|--------|------|
| `_fill_identity()` | 일반현황 식별표: 기업명 등 row-label 기준 채움 |
| `_fill_overview()` | 아이템개요 요약표: 문제인식/솔루션/팀구성 채움 |
| `_apply_paragraph_fills()` | 본문 앵커 문단 교체 + 새 불릿 문단 삽입 |
| `_apply_row_rewrites()` | 지정 테이블·행·열 직접 재기록 |
| `_apply_replacements()` | 전문 exact/prefix 일괄 치환 |
| `_blank_guides()` | `< ... >` 가이드 문구 빈칸 처리 |
| `_blank_residual()` | RESIDUAL_RE 매칭 더미 토큰 정리 |
| `scan_residual()` | 저장 후 잔여 더미 검증 스캔 |
| `finalize()` | 전체 파이프라인 실행 → report dict 반환 |

설계 원칙: 도메인 데이터(plan)는 외부 주입, 엔진에 기업 정보 하드코딩 없음.
